Informações sobre Função VAR e @media

ONLY PRINT:

Nesta propriedade podemos definir o que vai aparecer na tela quando a pessoa vai imprimir.

Sintaxe:
@media only print (condição) {atributo}

Podemos remover algumas coisas na tela, para fazer isso, usamos o display: none no atributo de uma tag, sendo assim, vai aparecer na tela e não na imprenssão. Um exemplo, na impressão de boletos.

ONLY SCREEN:

Nesta propriedade vamos definir o que vai aparecer na tela, isto é, coisas que vão aparecer no navedor, mas quando for imprimir pode aparecer outras coisas.

Sintaxe:
@media only screen (condição) {atributo}

Também podemos remover algumas tags usando o display:none em suas propriedades.

ORIENTATION:LANDSCAPE E PORTRAIK:

Esta propriedade do @media, ela vai trabalhar na proposição da tela, de um modo que quando a tela do navegador estiver com um número maior na horizontal vai aplicar uma determinada configuração, isso também se aplica na orizontal.

Sintaxe
@media (orientation: landscape ou portraik) {atributo}

landscape: horizontal
portraik: vertical

ASPECT-RADIO:

Define o tamanho da tela para aplicar algumas caracteristicas. Ela lembra bem padrão do @media com o min e max, mas o aspect-radio tem uma caracteristica única, sendo a forma com que o valor é implementado.

Sintaxe:
@media (aspect-radio: 1/1) {atributo}

Onde neste caso 1/1 significa Altura/Largura, portanto, temos um quadrado.

VAR:

O var ela é usada para criar um variável. Primeiramente, para criarmos uma variável devemos colocar -- e seguindo do seu nome, por fim, vamos adicionar o seu atributo. Feitos esses passos vamos ter uma variável.

Sintaxe:
--bg-color: red;

Podemos criar uma variável dentro de alguma div, mas ela funconará somente nela mesmo e nos seus filhos. E usamos o :root{ } global.

Degradê

O degradê vai dar um melhor efeito visual para o seu site. Podemos usa-lo de uma maneira fácil e universal para todos os navegadores.

Sintaxe:
background-color: linear-gradiente (direção, cor, cor)

Como temos muitos navegadores, alguns pode ser que não fucnione este degradê, para isso, vamos usar algumas funções complementares:

  • -moz-linear-gradiente (direção, cor, cor) : para o mozilia
  • -o-linear-gradiente (direção, cor, cor) : para o Opera
  • -webkit-linear-gradiente (direção, cor, cor) : para o Safari +6

E por fim, para colocarmos uma direção usamos o deg, signficando o grau, podemos colocar tando um valor positivo quanto um negativo. Também é possível usar a direção em inglês, como top, bottom, left, right, mas nem sempre vai funcionar em todo
